## 关键发现

- 需求来自 1.0.1 规划：默认开启 Codex 和 Claude Code 两个 Runtime。
- 设置中需要新增 Runtime 展示配置，允许用户选择展示哪些 Runtime。
- 至少必须选择一个 Runtime，避免 UI 没有可用数据范围。
- 用量趋势中的近 7 日和半年用量图表当前是静态图表，鼠标 hover 时缺少详情 tooltip。
- 趋势图 tooltip 需要帮助用户查看具体日期或节点的更细 token 信息，而不是只靠图形扫视。

## 开放问题

- 隐藏 Runtime 是否只控制展示，还是同时控制后台数据读取。
- 当前 Runtime 被隐藏后的自动切换与提示策略需要在 PRD 中明确。
- 近 7 日图表和半年图表 tooltip 的字段、交互方式和可访问性策略需要在 PRD 中明确。
